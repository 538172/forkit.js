forkit.js

- an animated GitHub Ribbon

This a concept for a more playful and animated GitHub ribbon. It was hacked together quickly so I won't vouch for browser compatibility.

The intention was to let you use the detached ribbon to drag down an iframe-embed of the repository on github.com but I neglected to check their X-Frame-Options until finishing most of the work.
